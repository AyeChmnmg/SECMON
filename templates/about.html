About SECMON





Version : {{version}}

SECMON is a real-time infosec monitoring tool that works from several sources.


It works with pollers that allow you to send emails with the latest CVE concerning your IT infrastructure as well as the latest cyber security information published on the web.


The Web UI allows you to manage more precisely the CVE module with multiple management functions.


SECMON (by Aubin Custodio - Guezone) is licensed under CC BY-NC-SA 4.0.


This license allows you to use SECMON, to improve it and to make it live by mentioning the author but without using it for commercial purposes. As the infosec monitoring process is quite difficult and time consuming, it should only be used to help companies and/or users to secure their IT infrastructure and to know the current cyberspace.
